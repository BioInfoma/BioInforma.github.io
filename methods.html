Pipeline Overview

From disease scope to
biomarker signature

Each step builds on the last. Students follow them in sequence. Every step has a defined goal, a set of databases or tools, and a concrete deliverable to be checked at the weekly meeting.

01
Foundation

Define the Disease and Biological Scope

Establish a shared understanding of PDAC, why early detection is difficult, and why exosomal circRNAs are promising biomarkers. Each student reads at least one paper on exosomes in cancer and one circRNA review before progressing.

Literature review ½–1 page written summary
02
Data Collection

Identify circRNAs Reported in Pancreatic Cancer

Search circRNA disease association databases and expression atlases for circRNAs reported as dysregulated in PDAC. For each candidate, collect: circRNA name, host gene, direction of change (up/down in PDAC), and the source paper.

circRNA disease databases Expression atlases Shared spreadsheet
03
Filtering

Filter for circRNAs Found in Exosomes

A circRNA can be dysregulated in tumor tissue but not released into blood. Cross-reference the candidate list against EV/exosome RNA databases. Keep only candidates with evidence of presence in extracellular vesicles — these are the ones that could become blood biomarkers.

EV RNA repositories Published EV datasets Refined candidate list
04
Interaction Mapping

Predict miRNAs That Interact with Each circRNA

Many circRNAs act as microRNA sponges: when a circRNA binds a miRNA, that miRNA becomes unavailable to suppress its target mRNAs. Use miRNA target prediction tools and circRNA–miRNA interaction resources to identify which microRNAs each candidate circRNA can sequester.

miRNA prediction tools circRNA–miRNA databases circRNA–miRNA interaction table
05
Target Identification

Identify Downstream mRNA Targets of the miRNAs

Retrieve predicted and experimentally validated mRNA targets for each miRNA identified in Step 4. Focus on genes involved in cancer, cell cycle regulation, metabolism, and inflammation. This completes the regulatory chain: circRNA → miRNA → mRNA → biological function.

miRNA target databases Validated interaction resources Complete regulatory chain table
06
Tumor Validation

Validate mRNA Targets Using TCGA-PAAD Tumor Data

Download mRNA expression and clinical data from TCGA-PAAD. Check whether predicted target genes are differentially expressed between PDAC tumors and normal pancreatic tissue. Remove genes that are not actually dysregulated in real patient data — this critical step filters out false computational predictions.

TCGA-PAAD R / DEG analysis Validated target gene list
07
Network Construction

Construct the ceRNA Regulatory Network

Build a circRNA–miRNA–mRNA regulatory network from all validated interactions. Nodes represent circRNAs, miRNAs, and mRNAs; edges represent regulatory relationships. Import into Cytoscape for visualization and topology analysis. Identify hub circRNAs — those connecting to many partners — as the strongest biomarker and functional candidates.

Cytoscape Network topology Hub circRNA identification
08
Functional Annotation

Perform Pathway and Functional Enrichment

Run pathway enrichment analysis on the validated mRNA target set using KEGG, Reactome, and MSigDB. Identify which cancer-related, immune, metabolic, and cell proliferation pathways are significantly enriched. A robust biomarker panel should connect to known cancer biology — not random genes.

KEGG Reactome MSigDB Enrichment table and figure
09
Clinical Relevance

Survival Analysis and Risk Stratification

Split TCGA-PAAD patients into high- and low-expression groups for each candidate circRNA. Perform Kaplan–Meier survival analysis using log-rank tests. Identify which circRNAs are significantly associated with overall survival. Combine the top candidates into a high-risk circRNA panel.

R survival package Kaplan–Meier plots Prognostic ranking
10
Signature Definition

Build the Final Biomarker Signature

Select a small, interpretable panel of circRNAs that satisfy all three evidence criteria: exosome-associated, participating in the ceRNA network, and linked to patient survival. Provide biological interpretation for each candidate. Smaller, biologically meaningful panels are far more clinically translatable than large gene lists.

Final biomarker panel Biological rationale per candidate
11
Reproducibility

Documentation and Reproducibility

Compile all clean, annotated scripts, a detailed methods summary, and a README explaining every analysis step. Archive all intermediate result tables. Reproducibility is not optional — a project that cannot be reproduced by a future researcher does not advance science.

Clean R scripts README Methods summary All intermediate tables
