Phase 1 · Weeks 1–4
Foundations — Biology & R Skills
4
Weeks
Week 1
Orientation + Molecular Biology Basics
Understand the project goal (exosomes + circRNAs + PDAC). Read two intro papers on exosomes and liquid biopsy. Install R and run basic syntax including printing, variables, and comments.
R Basics 1-page summary Literature
Week 2
Circular RNA Biology + Data Structures in R
Understand circRNAs and the ceRNA concept. Read 2–3 papers on circRNA biology and miRNA sponging. Learn R data types and structures: vectors, lists, matrices, and data frames.
R Data Structures circRNA review
Week 3
Pancreatic Cancer Context + Control Structures
Understand PDAC biology and the clinical early-detection problem. Read biomarker reviews. Learn R control structures: if/else, for loops, while loops.
R Loops & Logic PDAC review 1-page summary
Week 4
ceRNA Networks + Functions & Visualization
Understand regulatory network logic and the ceRNA framework. Write an R function. Create basic bar plots and scatter plots. Draw a ceRNA diagram on paper: circRNA → miRNA → mRNA.
ggplot2 ceRNA diagram R functions

Phase 1 — Meeting Goal

Explain exosomes as biomarkers in plain language
Explain what a circRNA is and why it is stable in blood
Explain early PDAC detection challenges
Demonstrate working R code in live review
Draw a ceRNA diagram from memory
Submit all three 1-page written summaries
Phase 2 · Weeks 5–8
Data Discovery & Candidate Identification
4
Weeks
Week 5
circRNA Discovery in PDAC
Search circRNA databases and PDAC literature. Build a shared spreadsheet of PDAC-associated circRNAs including circRNA name, host gene, direction of change (up/down), and source paper.
circRNA databases Spreadsheet output
Week 6
Exosome Filtering
Cross-reference circRNA candidates from Week 5 against EV/exosome RNA databases. Keep only those with evidence of presence in extracellular vesicles. Document every inclusion/exclusion decision with justification.
EV RNA databases Filtered list
Week 7
circRNA–miRNA Interaction Mapping
Use miRNA target prediction tools to identify microRNAs bound by each candidate circRNA. Record circRNA, miRNA, and binding evidence or prediction scores in an interaction table.
miRNA tools Interaction table
Week 8
miRNA–mRNA Target Mapping
Retrieve predicted and validated mRNA targets for each miRNA from Week 7. Filter for cancer-relevant genes: cell cycle, metabolism, inflammation. Compile the full circRNA–miRNA–mRNA interaction table.
Target databases Full interaction table

Phase 2 — Deliverables

Shared spreadsheet of PDAC circRNA candidates with references
Refined list of exosome-associated circRNAs with inclusion rationale
circRNA–miRNA interaction table with evidence scores
Complete circRNA–miRNA–mRNA regulatory chain table
Phase 3 · Weeks 9–12
Tumor Data Integration & Network Construction
4
Weeks
Week 9
TCGA-PAAD Data Familiarization
Download TCGA-PAAD mRNA expression and clinical data. Explore the dataset in R — sample sizes, variable types, missing data, and data distribution. Understand how the data is structured before any analysis.
TCGA-PAAD R exploration
Week 10
Expression Validation
Perform differential expression checks on predicted mRNA targets from Phase 2. Remove genes that are not dysregulated in real PDAC tumor data vs normal pancreatic tissue — this filters out false predictions from computational tools.
DEG analysis Validated target list
Week 11
ceRNA Network Construction
Create network input files from all validated circRNA–miRNA–mRNA interactions. Import into Cytoscape. Nodes represent circRNAs, miRNAs, and mRNAs; edges represent regulatory interactions from the interaction table.
Cytoscape Network figure
Week 12
Network Interpretation
Perform topology analysis. Extract hub circRNAs (highly connected nodes) and key mRNA targets. Write a short biological interpretation of network structure and identify the most promising biomarker candidates from network centrality.
Topology analysis Hub circRNAs

Phase 3 — Deliverables

Validated mRNA target list supported by TCGA-PAAD data
Complete ceRNA network figure from Cytoscape
List of hub circRNAs by network centrality
Short biological interpretation report
Phase 4 · Weeks 13–16
Pathway Enrichment & Survival Analysis
4
Weeks
Week 13
Pathway Enrichment Analysis
Run pathway enrichment on the validated mRNA target gene set using KEGG, Reactome, and MSigDB. Identify cancer-related, immune, metabolic, and cell proliferation pathways. A strong biomarker panel must connect to known cancer biology.
KEGG Reactome MSigDB
Week 14
Survival Analysis Setup
Load TCGA-PAAD clinical outcome data using the R survival package. Prepare patient groups by splitting samples into high- and low-expression groups for each candidate circRNA.
R survival pkg Patient groups
Week 15
Kaplan–Meier Survival Modeling
Generate Kaplan–Meier survival curves for each candidate circRNA. Apply log-rank tests to assess statistical significance. Rank circRNAs by their prognostic value and association with overall survival.
KM plots Log-rank test Prognostic ranking
Week 16
Final Biomarker Panel Selection
Select a small, clinically translatable circRNA panel. Each candidate must satisfy all three criteria: exosome-associated, participating in the ceRNA network, and significantly linked to patient survival. Provide biological rationale per candidate.
Final panel Biological rationale

Phase 4 — Deliverables

Pathway enrichment table and dot plot figure
Kaplan–Meier plots for all candidate circRNAs
Ranked list of prognostically significant circRNAs
Final circRNA biomarker panel with full justification
Phase 5 · Weeks 17–24
Writing, Figures & Final Output
8
Weeks
Week 17
Manuscript Figure Generation
Produce all six manuscript-ready figures using ggplot2 and ComplexHeatmap: conceptual framework, circRNA landscape heatmap, ceRNA network, pathway enrichment, KM survival curves, and summary model.
ggplot2 ComplexHeatmap 6 figures
Week 18
Methods Documentation
Each student writes one methods section. Compile all clean, annotated R scripts and a README file that allows a future researcher to reproduce every result from raw data without assistance.
README Clean scripts
Week 19
Interpretation & Limitations
Write the biological meaning of the findings and explicitly acknowledge limitations of in silico analysis: lack of experimental validation, database annotation gaps, and population-level vs. patient-level applicability.
Limitations Interpretation
Weeks 20–24
Manuscript Draft & Presentation
Draft the full manuscript. Prepare the presentation deck for internal team review. Compile the final internal project report combining all outputs from all five phases into a single cohesive document.
Manuscript draft Presentation deck Internal report

Final Project Deliverables — Every Researcher

Reproducible analysis pipeline with clean, annotated scripts
ceRNA network figure (Cytoscape) with hub annotation
Kaplan–Meier survival analysis and prognostic ranking
Final circRNA biomarker panel with biological rationale
Short interpretation and limitations report
Complete README for full reproducibility
